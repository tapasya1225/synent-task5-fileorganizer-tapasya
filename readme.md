# File Organizer 📂

A Python automation project that automatically organizes files into folders based on their file types.

---

## 🚀 Features

✅ Automatically sorts files  
✅ Creates folders dynamically  
✅ Supports images, videos, documents, music, and Python files  
✅ Organizes messy directories instantly  
✅ Beginner-friendly automation project  

---

## 🛠 Technologies Used

- Python
- os module
- shutil module

---

## 📁 Supported File Categories

| Category | File Types |
|---|---|
| Images | .jpg, .jpeg, .png, .gif |
| Documents | .pdf, .docx, .txt, .pptx |
| Videos | .mp4, .mkv, .avi |
| Music | .mp3, .wav |
| Python Files | .py |
